# TaskFlow EDU

A modern task management dashboard built for college students. Organize assignments, exams, projects, and more with a beautiful Kanban board, calendar view, and list view.

## Getting Started

First, install dependencies and run the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) with your browser to see the result.

## Features

- **Kanban Board** — drag-and-drop tasks between To Do, In Progress, and Done
- **Calendar View** — see all deadlines at a glance
- **List View** — sortable table of all tasks
- **Course Management** — organize tasks by course
- **Task Details** — subtasks, notes, attachments, progress tracking
- **Search & Filter** — find tasks quickly

## Tech Stack

- [Next.js 16](https://nextjs.org) with App Router
- [Tailwind CSS 4](https://tailwindcss.com)
- [TypeScript](https://typescriptlang.org)
- [Lucide Icons](https://lucide.dev)
